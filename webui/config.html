How to get a Page Access Token:
  1. Go to developers.facebook.com/apps and create or select an app
  2. Add the Facebook Login product to your app
  3. Go to Graph API Explorer
  4. Select your app, then click Get User Access Token
  5. Select permissions: pages_manage_posts, pages_read_engagement, pages_read_user_content, pages_manage_metadata
  6. Click Generate Access Token and authorize
  7. Use the Page Access Token from the dropdown (not the User token)
  8. For a long-lived token, exchange via the Access Token Debugger
